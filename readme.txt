== Description ==

A tiny plguin that allows you to create a stylish, eye-catching custom Admin Login Form. You can easily replace the default WordPress login style and build your own style without any coding knowledge.

### Key Features
* Custom Minimal Design
* Simple and Light weight
* Highly Customizable
* Cusotm Background color
* Custom Background image and style the login form
* Design custom Admin login without any coding knowledge

== Installation ==

It's really easy and super simple to install this plugin.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for `Stylish Admin Login Form`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download Advanced Accordion Block.
2.  Extract the `stylish-admin-login-form.zip` file. You'll get plugin files inside `stylish-admin-login-form` directory.
3.  Upload the plugin files to the `/wp-content/plugins/stylish-admin-login-form` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
